CAG Droid Race

Tävlingen går ut på att köra BB-8 runt banan på snabbaste tid. Man måste passera portarna i ordningen:

  • start
  • mellan
  • slut
i rätt ordning annars blir man diskad.

För att vara med skall man registrera en användare och sedan anmäla sig till tävlingen. När det är dags att köra får man en notis i appen.
